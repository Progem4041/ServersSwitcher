🚀 ServerSwitcher: The Fastest Way to Hop
Tired of the "Main Menu" loop? Every time you want to switch from a survival server to a creative test world, you have to disconnect, wait for the menu to load, find the server, and click again. ServerSwitcher fixes this by bringing your server list directly into the game.

✨ Key Features
Quick-Access Menu: A new "Switch Server" button added directly to your Escape (Pause) menu.

One-Click Reconnect: Lagging out? Need to refresh your proxy connection? Use the dedicated Reconnect button to rejoin your current server instantly.

Native Feel: Designed to look and feel like a vanilla part of Minecraft. No clunky or oversized UI.

Server List Integration: It automatically pulls all servers from your existing servers.dat file. No need to re-add them!

Purely Client-Side: You can use this on any server (Hypixel, local VPS, or private SMP) without needing any server-side plugins.
More information and download: https://modrinth.com/project/servers-switcher
